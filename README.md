## Cinnamon Theme

Cinnamon is my personal theme on Windows 11 — fast, instant, and clean. It keeps everything simple, smooth, and easy to use every day. I built it to feel focused, modern, and distraction-free while working or just chilling on my PC.

</div>

### Stack

This is a list of the various technologies used to build this theme:

| Category   | Technology Name |
| ---------- | --------------- |
| Core       | [**Zen Browser**](https://zen-browser.app/) |
| Core       | [**Zed Editor**](https://zed.dev/) |
| Core       | [**GlazeWM**](https://github.com/glzr-io/glazewm) |
| Core       | [**Flow Launcher**](https://flowlauncher.com/) |
| Decoration | [**MicaForEveryone**](https://github.com/MicaForEveryone/MicaForEveryone) |
| Decoration | [**F.lux**](https://justgetflux.com/) |
| Decoration | [**TranslucentTB**](https://apps.microsoft.com/detail/9pf4kz2vn4w9) |
| Decoration | [**tacky-border**](https://github.com/lukeyou05/tacky-borders) |
| Decoration | [**AutoHotkey**](https://www.autohotkey.com/) |
